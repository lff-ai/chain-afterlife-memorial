# chain-afterlife-memorial

一、 产品逻辑梳理 (User Flow)
进入页面： 用户看到一片深邃的星空（Three.js），太空中漂浮着许多形状各异的赛博风格墓碑。

漫游： 墓碑上刻着已归零的项目名字（如：LUNA, FTX, 3AC, BITCONNECT）。用户可以拖动视角查看。

交互（上香）：

点击某个墓碑，镜头拉近。

显示墓志铭（例如 FTX: "SBF was playing League of Legends while trading"）。

点击按钮 "Press F to Pay Respects"。

上链反馈：

发起一笔交易（0.001 Monad）。

Monad 特性展示： 交易瞬间确认，墓碑周围爆发出一阵金色粒子特效（代表“功德+1”），计数器数字跳动。

创建（可选）： 底部有一个按钮“埋葬新项目”，用户可以花钱立一个新的碑。
